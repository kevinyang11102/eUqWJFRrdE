# 酒店管理系统 java213

## 项目概述


## 技术栈

- **前端技术**：Vue.js、Element-UI
- **后端技术**：Spring Boot、MyBatis Plus
- **数据库**：MySQL
- **开发语言**：Java

## 功能模块

### 系统角色

- 管理员
- 用户

#### 管理员功能

- **登录**：管理员登录系统
- **用户管理**：管理用户信息
- **订单管理**：
  - 处理酒店预约订单
  - 查看订单列表
- **房间管理**：
  - 房间列表
  - 详情查询
  - 添加房间
  - 修改房间
- **评论列表**：查看用户评论

#### 用户功能

- **注册**：用户注册账户
- **登录**：用户登录系统
- **查询空余房间**：查询可预订的房间
- **房间预约**：预约酒店房间
- **我的**：
  - 查看预约订单

## 运行环境

- 前端：现代浏览器
- 后端：JDK 1.8 或更高版本、MySQL 5.7 或更高版本
